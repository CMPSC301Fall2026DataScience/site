## 📦 Datasets

### General Purpose Datasets

All datasets are available in the `materials/files/datasets/` directory:

- **sample_sales_data.csv**: E-commerce sales transactions (1000 rows)
- **student_performance.csv**: Academic performance data (500 students)
- **weather_data.csv**: Historical weather records (5 years)
- **social_media_posts.csv**: Social media text data (2000 posts)
- **stock_prices.csv**: Financial time series (10 companies, 2 years)
- **customer_reviews.csv**: Product reviews with ratings (3000 reviews)

### Specialized Datasets by Assignment

## 🆘 Need Help?

- **File not working?** Check that you've downloaded the correct version
- **Data issues?** Verify the file isn't corrupted (check file size)

## 📝 File Organization Tips

Recommended folder structure for your work:

```

├── labs/
│   ├── lab01/
│   ├── lab02/
│   └── ...
├── activities/
│   ├── activity01/
│   ├── activity02/
│   └── ...
├── projects/
│   ├── midterm/
│   └── final/
├── datasets/
│   └── (downloaded data files)
└── notes/
    └── (class notes)
```
